__SQL의 주요 명령어 분류__

__쿼리__ - DOL (데이터 질의어) : 데이터베이스에서 필요한 데이터를 검색하고 조회
SELECT : 조건에 맞는 데이터를 조회

__커맨드__ - DML (데이터 조작어) : 테이블 내의 데이터를 추가, 수정, 삭제         ----> CRUD
INSERT : 새 데이터 추가
UPDATE : 기존 데이터 수정
DELETE : 데이터 삭제

DDL (데이터 정의어) : 데이터베이스의 구조 만들거나 수정
CREATE : 새 테이블 생성
ALTER : 테이블 구조 변경

DCL (데이터 제어어) : 데이터베이스의 보안 및 권한 제어
GRANT : 권한 부여
